Projeto Pet Shop Pé de Paz – Fase 1

1. Objetivo Geral
Desenvolver a primeira versão do site institucional do Pet Shop Pé de Paz, utilizando apenas HTML, com foco na organização do conteúdo, apresentação de produtos e serviços e navegação básica.

2. Metas da Fase 1
Estruturar a página em cabeçalho, conteúdo principal e rodapé.
Inserir produtos das categorias Alimentação, Medicamentos e Produtos Diversos, com imagem, descrição e preço.
Descrever serviços oferecidos, incluindo valor e indicação de tele-busca.
Publicar no GitHub e GitHub Pages e manter o código organizado no repositório.

3. Acesso ao Projeto
Repositório: https://github.com/ITGaribaldic/FirstProjectUniversity
Página publicada: https://itgaribaldic.github.io/FirstProjectUniversity/

4. Estrutura da Página
Cabeçalho: Nome do estabelecimento e navegação interna.
Seções de Produtos e Serviços: Conteúdo exigido pela Fase 1, com imagens e informações básicas.
Rodapé: Dados fictícios de contato, resumo institucional e informações do autor.

5. Tecnologias Utilizadas
HTML5, Git/GitHub e GitHub Pages.
